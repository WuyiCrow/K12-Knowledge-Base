## 📋 数据格式

每个知识点包含以下字段：

```json
{
  "subject": "数学",
  "title": "函数概念",
  "content": "理解函数的定义、定义域、值域...",
  "source": "F-DataJS",
  "difficulty": "基础"
}
```

- **subject**: 学科名称
- **title**: 知识点标题
- **content**: 知识点内容描述
- **source**: 数据来源
- **difficulty**: 难度等级（基础/中等/提高）

## 🔧 导入工具

### 导入到 SQLite

```bash
cd import_tools
python import_to_sqlite.py
```

将生成 `knowledge.db` 数据库文件。

### 导入到向量数据库

```bash
cd import_tools
python import_to_vector_db.py
```

支持 ChromaDB / FAISS 等向量数据库。
